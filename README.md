# 0 Focus

A premium, customizable productivity timer designed to help you master your time and focus on what matters.

## ‚ú® Features

- **Smart Scheduling:** Automatically calculates work and break segments based on your target end time.
- **Pomodoro Mode:** Strict 25-minute focus intervals interspersed with 5-minute breaks.
- **Customizable Themes:**
  - Choose from **Light**, **Dark**, or fully **Custom** base themes.
  - Select your preferred **Accent Color** (Emerald, Blue, Rose, Amber, or any custom hex).
  - Define a specific **Break Color** for a distinct atmosphere during relaxation periods.
  - Customize background and surface colors for a truly personalized look.
- **Immersive Break Experience:** The entire application shifts color palette during break times to help you disconnect.
- **Sound Controls:** Toggle timer sounds and adjust volume with a custom slider.
- **Visual Timer:** A beautiful, breathing circular timer with satellite indicators and linear progress tracking.
- **Responsive Design:** Optimized for both desktop and mobile use.

## üì¶ Run Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/gek0v/0Focus.git
    cd 0Focus
    ```

2.  **Install dependencies:**
    ```bash
    pnpm install
    # or
    npm install
    ```

3.  **Start the development server:**
    ```bash
    pnpm dev
    # or
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    pnpm build
    ```

## üé® Customization Guide

Click the **Settings** (gear icon) in the header to access:
- **App Theme:** Switch between Light/Dark/Custom modes.
- **Accent Color:** Pick the primary color for buttons and highlights.
- **Break Color:** Set the color that takes over during breaks.
- **Sound:** Enable/disable sounds and set the volume.
